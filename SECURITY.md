## Reporting a Vulnerability

If you discover a security vulnerability in URBAN GARAGE POS, please follow these steps:

### 1. Do NOT disclose publicly

Please do not create a public GitHub issue or post about the vulnerability publicly until it has been addressed.

### 2. Report privately

Send an email to **security@urbangarage.com** with:
- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (if you have them)

### 3. Expect a response

We will acknowledge your email within 48 hours and provide a timeline for addressing the issue.

### 4. Coordinate disclosure

We will work with you to coordinate a responsible disclosure timeline that allows us to fix the issue before public disclosure.

## Security Best Practices

When using URBAN GARAGE POS:

- Keep your deployment up to date with the latest version
- Use HTTPS for all deployments
- Implement proper access controls for your POS terminals
- Regularly backup your data
- Use strong passwords for admin accounts
- Monitor your system for suspicious activity
